# Zingy Days Branding and Design Project

## Project Description: What it does and why
This project was created as a branding and design concept for Zingy Days, a small Irish fashion business. The aim of the project was to research the business, identify its target audience, and create a visual identity that reflects its values, style, and market position.

The project focuses on how Zingy Days can strengthen its branding through Irish cultural influence, modern fashion design, and visual consistency. The business was analysed as a handmade, family-owned Irish brand targeting younger audiences, particularly Gen Z and millennials. The final work explored how colour, typography, layout, and Celtic-inspired design elements could be used to create a strong and memorable brand identity.

## Purpose of the Project
The purpose of this project was to develop a branding direction for Zingy Days that would:
- reflect its Irish identity and handmade brand values
- appeal to a younger target audience
- create a more distinctive and professional visual style
- strengthen the brand’s presence across print and social media materials

This project also allowed me to apply design research, audience analysis, branding strategy, and layout design skills in a practical way.

## Project Contents
This branding project includes:
- brand research and audience analysis
- colour palette selection and explanation
- typography and design rationale
- Celtic and Irish visual inspiration
- poster design
- business card design
- booklet/spread layout work
- branding strategy and social media direction

## Design Approach
The visual style of the project was inspired by Irish culture, Celtic symbols, and traditional design motifs. The chosen colour palette included green, gold, and purple to represent Irish identity, luxury, and originality. Typography and decorative shapes were selected to create a bold and recognisable visual language that matched the business’s positioning as a modern Irish fashion brand.
